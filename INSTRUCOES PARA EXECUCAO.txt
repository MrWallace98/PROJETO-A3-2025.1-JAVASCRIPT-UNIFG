# Sistema de Gestão de Frotas

Sistema web para cadastro e controle de motoristas, carros e eventos de uso.

## Tecnologias
- Node.js (backend)
- MySQL (banco de dados)
- HTML, CSS, JS puro (frontend)

## Como rodar
1. Crie o banco de dados (veja o arquivo sql/banco.sql).
2. Atualize o arquivo db.js com seu usuário e senha do MySQL.
3. No terminal, rode:  
   npm install express mysql2 cors
4. Inicie o backend:  
   node app.js
5. Abra o index.html no navegador.
